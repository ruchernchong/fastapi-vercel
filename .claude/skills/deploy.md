# deploy

Deploy the FastAPI application to Vercel.

## Instructions

1. Check if there are any uncommitted changes using `git status`
2. If there are uncommitted changes, ask the user if they want to commit them first
3. Deploy to Vercel preview environment using: `vercel`
4. Ask the user if they want to deploy to production instead
5. If deploying to production, use: `vercel --prod`
6. Display the deployment URL returned by Vercel
7. Remind the user that the app is configured for the Singapore region (sin1)
